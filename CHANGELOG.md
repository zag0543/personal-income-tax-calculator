#### 1. 扣除项体检诊断系统
**来源**：用户痛点 - 不知道自己是否有遗漏的扣除项可以填报

**功能说明**：
- 智能问卷采集用户家庭、住房、教育、健康等关键信息
- 自动诊断九项扣除项（7项专项附加 + 个人养老金 + 税优健康险）
- 生成个性化体检报告，清晰展示可能遗漏的扣除项
- 计算潜在节税金额，给出具体操作指引

**实现位置**：
- `scripts/deduction_checkup.py` 新增完整脚本
- 支持交互式问卷和JSON非交互两种模式
- 报告输出包含：健康总览、可能遗漏项、行动清单、材料清单

**九项扣除诊断项**：
| 扣除项 | 诊断规则 | 预估节税 |
|--------|----------|----------|
| 子女教育 | 有正接受学历教育的子女 | 1,200-3,600元/年 |
| 3岁以下婴幼儿照护 | 有3岁以下子女 | 1,200-3,600元/年 |
| 继续教育 | 学历在读/职业资格证书 | 480-1,080元/年 |
| 大病医疗 | 医保内自付超1.5万 | 视情况而定 |
| 住房贷款利息 | 首套房贷款 | 1,200-3,600元/年 |
| 住房租金 | 一线城市租房 | 1,800-3,600元/年 |
| 赡养老人 | 父母满60岁 | 1,800-10,800元/年 |
| 个人养老金 | 已开通账户缴存 | 360-5,400元/年 |
| 税优健康险 | 已购买税优险 | 72-720元/年 |

**交互式问卷流程**：
```
第一步：基本信息（年收入范围）
第二步：家庭与住房（子女、老人、住房情况）
第三步：教育与健康（继续教育、资格证书、大病医疗）
第四步：保障类支出（个人养老金、税优健康险）
第五步：工作与租金（城市类型、户籍情况）
```

## v1.1.2 重要修复（2026-04-24）

### 🐛 年终奖计算逻辑修复
- **修正年终奖单独计税计算错误**：原代码将年终奖总额错误套用月度税率表
- 正确算法：用月平均额查税率，然后用总额计算税额
- 公式：税额 = 年终奖总额 × 税率 - 速算扣除数
- 验证：156,000元年终奖，月平均13,000元，适用20%税率，正确税额29,790元

#### 2. 非交互式计算模式 ⭐
**来源**：批量计算需求
**功能说明**：
- `questionnaire.py` 新增 `--json-input` 参数
- 支持通过命令行直接传入JSON数据进行计算
- 无需交互，适合程序调用和批量处理
- 新增 `--output-json` 参数，输出JSON格式结果

**使用方法**：
```bash
# 交互式模式（保持原有功能）
python questionnaire.py

# 非交互式JSON输入
python questionnaire.py --json-input '{"monthly_salary": 30000, "bonus": 50000}'

# 输出JSON格式结果
python questionnaire.py --json-input '{"monthly_salary": 30000}' --output-json
```
