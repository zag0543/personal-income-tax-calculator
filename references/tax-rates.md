## 三、全年一次性奖金计税税率表

适用于：全年一次性奖金单独计税（不并入综合所得）

**计税方法：**
1. 将全年一次性奖金除以12，得到月度应税所得
2. 根据月度应税所得找对应税率和速算扣除数
3. 计算应纳税额：`奖金金额 × 税率 - 速算扣除数`

| 月度应税所得 | 对应税率 | 速算扣除数 |
|:------------:|:--------:|:----------:|
| 不超过3,000元的 | 3% | 0 |
| 超过3,000元至12,000元的部分 | 10% | 210 |
| 超过12,000元至25,000元的部分 | 20% | 1,410 |
| 超过25,000元至35,000元的部分 | 25% | 2,660 |
| 超过35,000元至55,000元的部分 | 30% | 4,410 |
| 超过55,000元至80,000元的部分 | 35% | 7,160 |
| 超过80,000元的部分 | 45% | 15,160 |

**示例：**
- 年终奖36,000元：36000÷12=3000 → 税率3% → 税额=36000×3%-0=1,080元
- 年终奖60,000元：60000÷12=5000 → 税率10%，扣除210 → 税额=60000×10%-210=5,790元

## 六、应纳税所得额计算

### 综合所得
```
应纳税所得额 = 收入 - 60,000 - 专项扣除 - 专项附加扣除 - 其他扣除 - 依法确定的其他扣除
```

### 工资薪金（月度）
```
应纳税所得额 = 税前工资 - 5,000 - 专项扣除 - 专项附加扣除 - 其他扣除
```

## 七、速算扣除数说明

速算扣除数是为了简化计算而设置的固定数值。

**为什么需要速算扣除数？**
- 累进税率下，每一档需要分段计算，比较繁琐
- 速算扣除数可以将分段计算简化为一步计算
- 保证计算结果与分段计算完全一致

**计算示例：**
假设应纳税所得额为100,000元

分段计算：
- 36,000 × 3% = 1,080
- (100,000 - 36,000) × 10% = 6,400
- 合计：7,480元

使用速算扣除数：
- 100,000 × 10% - 2,520 = 7,480元 ✓
