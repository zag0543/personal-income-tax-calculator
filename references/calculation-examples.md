# 个人所得税计算示例

## 示例一：普通工薪族（月薪1.2万，年终奖2万）

### 基本信息
- 年度：2024年
- 月税前工资：12,000元（12个月共144,000元）
- 年终奖：20,000元（选择单独计税）
- 社保公积金个人部分：每月1,800元（其中公积金800元）
- 专项附加扣除：子女教育2,000元/月 + 住房租金1,500元/月
- 无其他扣除
- 单位全年代扣代缴：6,000元

### 计算过程

#### 1. 月度累计预扣预缴（1月）
```
累计收入 = 12,000元
累计减除费用 = 5,000元
累计专项扣除 = 1,800元
累计专项附加扣除 = (2,000 + 1,500) = 3,500元
累计应纳税所得额 = 12,000 - 5,000 - 1,800 - 3,500 = 1,700元
累计应纳税额 = 1,700 × 3% - 0 = 51元
```

#### 2. 月度累计预扣预缴（6月）
```
累计收入 = 12,000 × 6 = 72,000元
累计减除费用 = 5,000 × 6 = 30,000元
累计专项扣除 = 1,800 × 6 = 10,800元
累计专项附加扣除 = 3,500 × 6 = 21,000元
累计应纳税所得额 = 72,000 - 30,000 - 10,800 - 21,000 = 10,200元
累计应纳税额 = 10,200 × 3% - 0 = 306元
```

#### 3. 年度汇算清缴
```
综合所得收入 = 144,000元
减除费用 = 60,000元
专项扣除 = 1,800 × 12 = 21,600元
专项附加扣除 = 3,500 × 12 = 42,000元
其他扣除 = 0元
应纳税所得额 = 144,000 - 60,000 - 21,600 - 42,000 = 20,400元
应纳税额 = 20,400 × 3% - 0 = 612元

年终奖单独计税：
20,000 ÷ 12 = 1,666.67元 → 税率3% → 速算扣除数0
年终奖税额 = 20,000 × 3% - 0 = 600元

合计应纳税额 = 612 + 600 = 1,212元
```

#### 4. 对比结果
```
单位代扣代缴 = 6,000元
应纳税额 = 1,212元
应退税额 = 6,000 - 1,212 = 4,788元
```

### 结果
✅ **应退税 4,788元**

## 示例二：中高收入者（月薪3.5万，年终奖10万）

### 基本信息
- 年度：2024年
- 月税前工资：35,000元（12个月共420,000元）
- 年终奖：100,000元（选择单独计税）
- 社保公积金个人部分：每月5,500元（含职业年金500元）
- 专项附加扣除：
  - 子女教育2,000元
  - 赡养老人3,000元（独生子女）
  - 住房贷款利息1,000元
- 其他扣除：个人养老金12,000元
- 单位全年代扣代缴：85,000元

### 计算过程

#### 1. 综合所得收入
```
工资薪金 = 35,000 × 12 = 420,000元
```

#### 2. 扣除项目
```
基本减除费用 = 60,000元

专项扣除：
- 社保公积金 = (5,500 - 500) × 12 = 60,000元
- 职业年金 = 500 × 12 = 6,000元
- 合计 = 66,000元

专项附加扣除：
- 子女教育 = 2,000 × 12 = 24,000元
- 赡养老人 = 3,000 × 12 = 36,000元
- 住房贷款 = 1,000 × 12 = 12,000元
- 合计 = 72,000元

其他扣除：
- 个人养老金 = 12,000元
```

#### 3. 应纳税所得额
```
应纳税所得额 = 420,000 - 60,000 - 66,000 - 72,000 - 12,000 = 210,000元
```

#### 4. 综合所得应纳税额
```
适用税率：第3级（144,000-300,000），20%
速算扣除数：16,920元
应纳税额 = 210,000 × 20% - 16,920 = 25,080元
```

#### 5. 年终奖单独计税
```
100,000 ÷ 12 = 8,333.33元
适用税率：第2级，10%，速算扣除数210
税额 = 100,000 × 10% - 210 = 9,790元
```

#### 6. 合计应纳税额
```
合计 = 25,080 + 9,790 = 34,870元
```

#### 7. 对比结果
```
单位代扣代缴 = 85,000元
应纳税额 = 34,870元
应退税额 = 85,000 - 34,870 = 50,130元
```

### 结果
✅ **应退税 50,130元**

## 示例三：需要补税的情况（月薪5万，换工作）

### 基本信息
- 年度：2024年
- 上半年月薪：30,000元（6个月）
- 下半年月薪：70,000元（6个月）
- 无年终奖
- 社保公积金个人部分：每月7,500元
- 专项附加扣除：赡养老人3,000元
- 其他扣除：企业年金6,000元
- 单位代扣代缴：约45,000元（按各自单位预扣）

### 计算过程

#### 1. 综合所得收入
```
上半年：30,000 × 6 = 180,000元
下半年：70,000 × 6 = 420,000元
合计 = 600,000元
```

#### 2. 扣除项目
```
基本减除费用 = 60,000元
专项扣除 = 7,500 × 12 = 90,000元
专项附加扣除 = 3,000 × 12 = 36,000元
其他扣除 = 6,000元
合计扣除 = 192,000元
```

#### 3. 应纳税所得额
```
应纳税所得额 = 600,000 - 192,000 = 408,000元
```

#### 4. 综合所得应纳税额
```
适用税率：第4级（300,000-420,000），25%
速算扣除数：31,920元
应纳税额 = 408,000 × 25% - 31,920 = 70,080元
```

#### 5. 对比结果
```
单位代扣代缴 = 45,000元
应纳税额 = 70,080元
应补税额 = 70,080 - 45,000 = 25,080元
```

### 结果
⚠️ **应补税 25,080元**

## 示例四：大病医疗扣除计算

### 场景
纳税人2024年发生大病医疗，自付金额80,000元

### 计算
```
可扣除金额 = (80,000 - 15,000) × 60% = 39,000元
```

### 注意事项
- 需要在年度汇算清缴时自行填报
- 需保留医疗费用票据
- 可由本人或配偶扣除；未成年子女由父母扣除

## 示例五：全年一次性奖金两种计税方式对比

### 场景
年终奖36,000元，月工资20,000元，其他扣除3,000元/月

#### 方式一：单独计税
```
年终奖税额 = 36,000 × 3% - 0 = 1,080元

工资薪金年度应纳税：
收入 = 240,000元
扣除 = 60,000 + 3,000×12 = 96,000元
应纳税所得额 = 144,000元
税额 = 144,000 × 10% - 2,520 = 11,880元

合计 = 1,080 + 11,880 = 12,960元
```

#### 方式二：并入综合所得
```
收入合计 = 240,000 + 36,000 = 276,000元
扣除 = 96,000元
应纳税所得额 = 180,000元
税额 = 180,000 × 20% - 16,920 = 19,080元
```

### 对比
| 计税方式 | 税额 |
|----------|------|
| 单独计税 | 12,960元 |
| 并入综合所得 | 19,080元 |
| **差额** | **6,120元** |

### 结论
本例中，**单独计税更优惠**，可少缴6,120元

## 常见计算错误提醒

### ❌ 错误1：将年终奖全额直接乘税率
正确做法：年终奖÷12后找税率，再计算税额

### ❌ 错误2：同时扣除房贷和租金
正确做法：只能选择其中一项

### ❌ 错误3：忘记将月度扣除乘以12
正确做法：确认是月度扣除还是年度扣除

### ❌ 错误4：重复扣除子女教育
正确做法：夫妻双方合计扣除不超过2,000元/孩/月

### ❌ 错误5：大病医疗直接扣除全部自付额
正确做法：先减去15,000元门槛，再按60%计算
