# 个人所得税专项附加扣除政策文件摘要

## 一、子女教育

**政策依据：** 《个人所得税专项附加扣除暂行办法》第二章

### 扣除标准
- **扣除范围**：学历教育（学前教育至学历教育終了）
  - 学前教育：3岁以上
  - 学历教育：义务教育（小学、初中）、高中阶段教育（普通高中、中等职业）、高等教育（大学专科、大学本科、硕士研究生、博士研究生）
- **扣除标准**：2,000元/孩/月
- **扣除主体**：父母可以选择由其中一方按扣除标准的100%扣除，也可以由双方分别按扣除标准的50%扣除

### 常见问题
| 问题 | 答案 |
|------|------|
| 3岁以下可以扣除吗？ | 不可以。3岁以下属于3岁以下婴幼儿照护项目 |
| 高中毕业可以继续扣除吗？ | 不可以。学历教育不包括大学以上的学历教育 |
| 子女在境外读书可以扣除吗？ | 可以。学历教育包括中国境外学历教育 |
| 多个子女可以叠加扣除吗？ | 可以。每个子女都按标准扣除 |

## 二、继续教育

**政策依据：** 《个人所得税专项附加扣除暂行办法》第三章

### 扣除标准
| 类型 | 条件 | 扣除标准 | 扣除期限 | 扣除主体 |
|------|------|----------|----------|----------|
| 学历（学位）继续教育 | 在中国境内接受学历（学位）继续教育 | 400元/月 | 学历教育结束当月（最长48个月） | 本人 |
| 技能人员职业资格继续教育 | 取得相关证书当年 | 3600元/年 | 取得证书当年 | 本人 |
| 专业技术人员职业资格继续教育 | 取得相关证书当年 | 3600元/年 | 取得证书当年 | 本人 |

### 职业资格目录
参见人力资源社会保障部公布的《国家职业资格目录》
- 常见准入类：教师资格、医生资格、注册会计师等
- 常见水平评价类：会计专业技术资格、税务师等

### 常见问题
| 问题 | 答案 |
|------|------|
| 自考、成考可以扣除吗？ | 可以。属于学历继续教育 |
| 多个学历教育可以叠加吗？ | 不可以。同一个人同一学历阶段只能享受一次 |
| 考证没通过可以扣除吗？ | 不可以。必须实际取得证书 |
| 技能培训可以扣除吗？ | 不可以。必须是职业资格继续教育 |

## 三、大病医疗

**政策依据：** 《个人所得税专项附加扣除暂行办法》第四章

### 扣除标准
- **扣除范围**：在一个纳税年度内，纳税人发生的与基本医保相关的医药费用支出
- **扣除金额**：自付部分超过15,000元的据实扣除，上限80,000元/年
- **计算公式**：`（医保报销后自付金额 - 15,000）× 60%`
- **扣除主体**：本人或配偶扣除；未成年子女由父母一方扣除

### 操作方式
- 由纳税人在次年3月1日至6月30日汇算清缴时自行申报
- 需提供医疗费用票据或在医保服务平台查询

### 常见问题
| 问题 | 答案 |
|------|------|
| 只有住院才能扣除吗？ | 不是。门诊、住院都可以，只要与基本医保相关 |
| 商业保险可以扣除吗？ | 不可以。必须是基本医保相关费用 |
| 家人看病可以扣除吗？ | 可以。在年度汇算时填写配偶或未成年子女信息 |
| 自费部分可以全部扣除吗？ | 不可以。需先减去15,000元门槛，再按60%扣除 |

## 四、住房贷款利息

**政策依据：** 《个人所得税专项附加扣除暂行办法》第五章

### 扣除标准
- **扣除范围**：纳税人本人或配偶单独或共同使用商业银行或住房公积金个人住房贷款为本人或配偶购买中国境内住房
- **房屋条件**：首套住房贷款利息支出
- **扣除标准**：1,000元/月（120,000元/年）
- **扣除期限**：贷款合同约定开始还款的当月至贷款全部归还或贷款合同终止的当月（最长240个月）
- **扣除主体**：经夫妻双方约定，可以选择由其中一方扣除

### 常见问题
| 问题 | 答案 |
|------|------|
| 如何认定"首套住房"？ | 以是否享受首套房贷款利率认定 |
| 婚后购买算首套吗？ | 婚后以家庭为单位认定，如此前无房贷记录可按首套 |
| 已还清贷款还能扣除吗？ | 不能。贷款合同终止后停止扣除 |
| 婚前各自买房可以吗？ | 可以各自扣除750元/月，或选择一方扣除1000元/月 |

## 五、住房租金

**政策依据：** 《个人所得税专项附加扣除暂行办法》第六章

### 扣除标准
| 城市级别 | 适用城市 | 扣除标准 |
|-----------|----------|----------|
| 一类城市 | 直辖市、省会城市、计划单列市等 | 1,500元/月 |
| 二类城市 | 市辖区户籍人口超过100万的城市 | 1,100元/月 |
| 三类城市 | 市辖区户籍人口不超过100万的城市 | 800元/月 |

### 主要城市分类（参考）
| 城市级别 | 城市示例 |
|----------|----------|
| 一类（1,500元/月） | 北京、上海、广州、深圳、成都、重庆、杭州、武汉、西安、郑州、长沙、沈阳、青岛、宁波、厦门等 |
| 二类（1,100元/月） | 济南、昆明、合肥、长春、福州、哈尔滨、石家庄、苏州、无锡、南昌、贵阳、太原、兰州等 |
| 三类（800元/月） | 其他城市 |

### 扣除条件
1. 主要工作城市没有自有住房
2. 签订住房租赁合同
3. 在主要工作城市租房

### 常见问题
| 问题 | 答案 |
|------|------|
| 和房贷可以同时扣除吗？ | 不能。二者只能选其一 |
| 租房和住房在不同城市可以吗？ | 可以。以主要工作城市的租房情况为准 |
| 短租可以吗？ | 可以。按实际租赁时间扣除 |
| 夫妻同地工作怎么扣除？ | 只能由签订租赁合同的一方扣除 |

## 六、赡养老人

**政策依据：** 《个人所得税专项附加扣除暂行办法》第七章

### 扣除标准
| 身份 | 扣除标准 | 分摊方式 |
|------|----------|----------|
| 独生子女 | 3,000元/月 | 全部由本人扣除 |
| 非独生子女 | 最高1,500元/月/人 | 兄弟姐妹分摊，每人分摊不超过1,500元/月 |

### 被赡养人条件
- 年满60周岁的父母
- 子女均已去世的年满60周岁的祖父母、外祖父母

### 常见问题
| 问题 | 答案 |
|------|------|
| 继父母可以扣除吗？ | 不可以。被赡养人指生父母、养父母、继父母 |
| 祖父母由孙子赡养可以吗？ | 可以。子女均已去世的祖父母、外祖父母可由孙辈赡养 |
| 公婆/岳父母可以扣除吗？ | 不可以。只能扣除自己的父母 |
| 父母退休了还能扣除吗？ | 可以。只要年满60周岁即可 |

---

## 七、3岁以下婴幼儿照护

**政策依据：** 《关于设立3岁以下婴幼儿照护个人所得税专项附加扣除的通知》

### 扣除标准
- **扣除范围**：纳税人照护3岁以下婴幼儿的相关支出
- **扣除标准**：2,000元/孩/月
- **扣除主体**：父母可以选择由其中一方按扣除标准的100%扣除，也可以由双方分别按扣除标准的50%扣除

### 常见问题
| 问题 | 答案 |
|------|------|
| 3岁以下包含3岁生日当天吗？ | 包含。从出生到3岁生日当月均可扣除 |
| 双胞胎怎么扣？ | 每个婴幼儿都按2,000元/月扣除，即4,000元/月 |
| 祖父母看护可以扣除吗？ | 可以。但必须由子女（父母）作为扣除主体 |
| 早教费用可以扣除吗？ | 不可以。仅限照护支出，托育费、早教费等不在范围内 |

## 八、个人养老金

**政策依据：** 《关于个人养老金有关个人所得税政策的公告》

### 扣除标准
- **缴费扣除**：个人向个人养老金资金账户的缴费，在个税税前据实扣除
- **年度上限**：12,000元/年
- **投资环节**：计入个人养老金资金账户的投资收益暂不征收个人所得税
- **领取环节**：个人领取的个人养老金，不并入综合所得，单独按照3%的税率计算缴纳个人所得税

### 操作方式
1. 在银行开立个人养老金账户
2. 通过个税APP的"个人养老金扣除信息管理"扫码录入缴费信息
3. 年度汇算时自动扣除

## 九、税优健康险

**政策依据：** 《关于将商业健康保险个人所得税试点政策推广到全国范围实施的通知》

### 扣除标准
- **产品类型**：符合规定的商业健康保险产品
- **扣除标准**：每年不超过2,400元（200元/月）
- **扣除方式**：税前扣除，不需要年度汇算

### 税优健康险特征
- 通常在保单上标注"税优健康险"字样
- 可带病投保，保证续保
- 不得设置免赔额
